# Twitter analyzer

## Tools

* SparkML
* Spark streaming
* Akka: http, stream, actor

## Important notice

Don't forget to install [git-lfs](https://help.github.com/articles/installing-git-large-file-storage/#platform-linux) before cloning this repo

## How to run?

1. Run **ModelsTrainer**
1. Obtain creditionals for twitter [here](apps.twitter.com)
1. Fullfill `twitter4j.oauth.consumerKey`, `twitter4j.oauth.consumerSecret`, `twitter4j.oauth.accessToken`, `twitter4j.oauth.accessTokenSecret` into environment variables (for instance in idea run configuration window)
1. Run **Main**
1. Open **frontend/index.html**
1. Press enter in idea
1. Enjoy tsunami of logs and keep an eye to the index.html
